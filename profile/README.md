# Mavunta Developers

One API for payments, wallets, and crypto settlement. Accept M-Pesa, Airtel Money, card, PayPal, and stablecoins; settle to USDT or USDC.

**Docs:** https://developers.mavunta.com · **API reference:** https://developers.mavunta.com/pay · **Status:** https://www.mavunta.com/status

## Official SDKs

| Language | Install | Repo |
|---|---|---|
| Node.js / TypeScript | `npm install mavunta` | [mavunta-js](https://github.com/mavunta/mavunta-js) |
| CLI (bin `mavunta`) | `npm install -g mavunta-cli` | [mavunta-js](https://github.com/mavunta/mavunta-js) |
| Browser checkout | `npm install @mavunta/checkout-js` | [mavunta-js](https://github.com/mavunta/mavunta-js) |
| React | `npm install @mavunta/react` | [mavunta-js](https://github.com/mavunta/mavunta-js) |
| Python | `pip install mavunta` | [mavunta-python](https://github.com/mavunta/mavunta-python) |
| PHP | `composer require mavunta/mavunta-php` | [mavunta-php](https://github.com/mavunta/mavunta-php) |
| Laravel | `composer require mavunta/laravel-mavunta` | [laravel-mavunta](https://github.com/mavunta/laravel-mavunta) |
| Go | `go get github.com/mavunta/mavunta-go` | [mavunta-go](https://github.com/mavunta/mavunta-go) |
| Java | `com.mavunta:mavunta-java` (Maven Central rollout) | [mavunta-java](https://github.com/mavunta/mavunta-java) |

Sandbox first: `cwk_test_` keys on `sandbox-api.mavunta.com/v1` never move real money.
